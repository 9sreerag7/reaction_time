# Reaction Time Game 

A simple frontend game built using **HTML, CSS, and JavaScript** to measure how fast a user reacts to a visual cue.

## How It Works
- Click **Start**
- Wait until the box turns **green**
- Click as fast as you can
- Your reaction time is displayed in milliseconds

## Concepts Used
- DOM manipulation
- Event listeners
- setTimeout
- JavaScript functions
